# Prima App Angular Demo

Versione demo di un’app Angular, creata per esercitarsi con componenti, routing e gestione dati locali.

## 🛠 Tech Stack
- Angular 13
- TypeScript
- HTML5 & CSS3
- JavaScript
- Git & GitHub

## 🚀 Funzionalità
- Struttura a componenti Angular
- Routing tra pagine
- Gestione dati locali con db.json
- UI semplice e responsive
